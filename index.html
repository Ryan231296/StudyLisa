Apa yang Akan Kita Buat?

GAMBARAN BESAR
🚀 Produk Akhir: Aplikasi WebCraft
Kita akan membangun sebuah aplikasi Android (.APK) bernama WebCraft — sebuah website builder berbasis mobile. Pengguna bisa membuat halaman web hanya dengan menekan tombol, memilih komponen, mengatur tampilan, lalu mengekspornya sebagai file HTML yang langsung bisa diunggah ke hosting.
🎨
Visual Editor
Tambah & susun elemen website secara visual
📦
Template Siap
Landing page, portofolio, blog — tinggal pilih
⬇️
Export HTML
Hasilkan file HTML/CSS bersih siap upload
💾
Simpan Proyek
Proyek tersimpan di HP, bisa dilanjut kapan saja
👁️
Live Preview
Lihat tampilan website real-time di dalam app
🌗
Dark / Light Mode
Pilihan tema tampilan aplikasi
💡 Mengapa ini bermanfaat? Jutaan orang ingin punya website tapi tidak bisa coding. Aplikasi yang kamu buat ini akan membantu mereka membangun website sendiri dari HP — tanpa perlu buka laptop atau belajar HTML.
📋 Apa yang Akan Kamu Pelajari
  • Cara setup environment pengembangan aplikasi mobile
  • Dasar-dasar React Native dan cara kerjanya
  • Membuat sistem navigasi antar halaman di aplikasi
  • Menyimpan dan membaca data dari penyimpanan lokal HP
  • Menghasilkan kode HTML dari data di dalam aplikasi
  • Membangun (build) APK dan cara mendistribusikannya
🧠 Cek Pemahaman: Apa itu file .APK?
✅ Benar! APK = Android Package Kit. Sama seperti .exe di Windows atau .dmg di Mac — ini adalah file yang kamu install untuk memasang aplikasi di Android.

Simpan & Load Proyek

PENYIMPANAN DATA
💾 AsyncStorage adalah cara menyimpan data di HP seperti "notepad permanen" — data tetap ada meskipun aplikasi ditutup. Mirip seperti localStorage di web browser.
src/utils/storage.js — Semua Fungsi Penyimpanan
import AsyncStorage from '@react-native-async-storage/async-storage'; const PROJECTS_KEY = '@webcraft_projects'; /** * Simpan satu proyek (tambah baru atau update yang sudah ada) */ export const saveProject = async (project) => { try { const existing = await loadAllProjects(); const index = existing.findIndex(p => p.id === project.id); if (index >= 0) { existing[index] = project; // Update proyek yang sudah ada } else { existing.unshift(project); // Tambah proyek baru di awal daftar } await AsyncStorage.setItem(PROJECTS_KEY, JSON.stringify(existing)); return true; } catch (error) { console.error('Error menyimpan proyek:', error); return false; } }; /** * Load semua proyek */ export const loadAllProjects = async () => { try { const data = await AsyncStorage.getItem(PROJECTS_KEY); return data ? JSON.parse(data) : []; } catch (error) { console.error('Error loading proyek:', error); return []; } }; /** * Load satu proyek berdasarkan ID */ export const loadProject = async (projectId) => { const projects = await loadAllProjects(); return projects.find(p => p.id === projectId) || null; }; /** * Hapus proyek berdasarkan ID */ export const deleteProject = async (projectId) => { try { const projects = await loadAllProjects(); const filtered = projects.filter(p => p.id !== projectId); await AsyncStorage.setItem(PROJECTS_KEY, JSON.stringify(filtered)); return true; } catch (error) { console.error('Error hapus proyek:', error); return false; } }; /** * Hapus semua proyek (reset total) */ export const clearAllProjects = async () => { await AsyncStorage.removeItem(PROJECTS_KEY); };

/** * Mengonversi array elemen dari editor menjadi halaman HTML lengkap * yang siap diunggah ke hosting manapun */ export function generateHTML(elements, options = {}) { const { projectName = 'Website Saya', primaryColor = '#007bff', fontFamily = 'system-ui, sans-serif', description = 'Dibuat dengan WebCraft' } = options; // Render setiap elemen menjadi tag HTML const renderElement = (el) => { switch (el.type) { case 'navbar': return ` <nav class="wc-navbar"> <div class="wc-container"> <a href="#" class="wc-brand">${projectName}</a> <div class="wc-nav-links"> <a href="#">Home</a> <a href="#">Tentang</a> <a href="#">Kontak</a> </div> <button class="wc-hamburger" onclick="toggleMenu()">☰</button> </div> </nav>`; Build File APK

JADIKAN APLIKASI NYATA
🚀 Kita gunakan EAS Build — layanan cloud Expo yang mem-build APK di server mereka. Kamu tidak perlu install Android Studio. Cukup jalankan satu perintah, tunggu 10–15 menit, dan APK siap didownload.
1
Konfigurasi app.json
Set nama, bundle ID, versi, dan icon aplikasi
2
Buat file eas.json
Tentukan tipe build: apk untuk distribusi langsung, app-bundle untuk Play Store
3
Jalankan perintah build
eas build --platform android --profile preview
4
Download APK
Setelah selesai (~15 menit), link download APK akan muncul di terminal dan email
app.json — Konfigurasi Aplikasi
{ "expo": { "name": "WebCraft", "slug": "webcraft-builder", "version": "1.0.0", "orientation": "portrait", "icon": "./assets/icon.png", "userInterfaceStyle": "dark", "splash": { "image": "./assets/splash.png", "resizeMode": "contain", "backgroundColor": "#05070d" }, "android": { "adaptiveIcon": { "foregroundImage": "./assets/adaptive-icon.png", "backgroundColor": "#05070d" }, "package": "com.namaKamu.webcraft", "versionCode": 1, "permissions": [ "READ_EXTERNAL_STORAGE", "WRITE_EXTERNAL_STORAGE" ] }, "plugins": ["expo-file-system"] } }
eas.json — Konfigurasi Build
{ "cli": { "version": ">= 5.0.0" }, "build": { "development": { "developmentClient": true, "distribution": "internal" }, "preview": { "android": { "buildType": "apk" // ← INI yang menghasilkan file .APK } }, "production": { "android": { "buildType": "app-bundle" // ← Untuk upload ke Play Store } } } }
Terminal — Perintah Build APK
# Pastikan sudah login dulu eas login # Inisialisasi proyek EAS (hanya perlu sekali) eas build:configure # BUILD APK — Ini yang utama! eas build --platform android --profile preview # Tunggu 10-20 menit. Setelah selesai, jalankan ini # untuk mendapat link download APK: eas build:list # Atau buka dashboard di browser: # https://expo.dev/accounts/[username]/projects/webcraft-builder/builds

Testing & Distribusi

LANGKAH TERAKHIR
✅ Checklist Testing Wajib
  • Install APK di 2-3 HP Android berbeda untuk test kompatibilitas
  • Buat proyek baru — tambah semua tipe elemen, edit isinya
  • Simpan proyek, tutup aplikasi, buka lagi — pastikan data masih ada
  • Export ke HTML — buka file di browser HP & komputer
  • Test mode portrait dan landscape (putar HP)
  • Coba hapus proyek dari Home Screen — pastikan benar terhapus
  • Ganti nama app, icon, dan splash screen jadi custom
  • Preview website di dalam app tampil dengan benar

📤 Cara Distribusi APK

🔗
Google Drive
Upload APK, buat link publik, bagikan ke siapapun. Paling mudah!
🐙
GitHub Releases
Profesional. Upload sebagai Release — ada version history otomatis
🏪
Google Play Store
Untuk distribusi publik besar. Daftar $25 sekali, build app-bundle
⚠️ Izin Install APK di Android: Saat user pertama kali install APK dari luar Play Store, Android akan minta izin "Install from unknown sources". Ini normal — user perlu mengaktifkannya di Settings → Security.

  • Tambah lebih banyak komponen: galeri foto, maps, form input, timer

  • Tambah fitur edit warna, font, dan ukuran per elemen
  • Integrasi upload ke hosting langsung dari aplikasi
  • Tambah fitur undo/redo untuk editor
